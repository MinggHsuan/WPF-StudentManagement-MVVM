# 學生管理系統(Student Management System)

## 📖 專案描述
- 這是一個使用C# MVP架構開發的學生管理系統，小型的WPF專案。
- 想要做到主要功能運作良好及邏輯與介面分離，但畫面陽春的情況下，刪除整個Xaml檔，由AI重新生成一個任何風格的介面Xaml檔，能夠直接套用且功能都正常使用。

## 🚀 主要功能
- 學生基本資料 CRUD（新增、查詢、修改、刪除）


## 🏗️ 技術架構
- **語言**：C#
- **UI框架**：WPF
- **框架**：.NET Framework
- **架構模式**：MVVM + MVP
- **資料處理**：自製CSV Reader/Writer Library
